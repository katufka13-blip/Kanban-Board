Канбан-доска

Разработка React-приложение для решения задач в формате канбана, с созданием компонентов, логикой авторизации, регистрации и задачами управления (добавление, удаление, редактирование).
Управление состоянием через Context API.
Обеспечено форматирование кода с помощью Prettier и Eslint для единообразия.
Реализована интеграция с API для управления данными и пользователями.
